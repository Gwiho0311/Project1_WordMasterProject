WordCRUD Program
================
단어를 추가하고 읽고 업데이트하고 삭제하는 등의 다양한 기능을 수행할 수 있는 프로그램이다.

Files
-----
Dictionary.txt 파일은 단어들이 저장되는 텍스트 파일이다.
